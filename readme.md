# ShareBot – Indian Stocks & Forex Learning Chatbot

This is a simple educational chatbot built with **Python** and **Streamlit** to help beginners understand:

- Indian stock market basics (NSE/BSE)
- Fundamental concepts like PE, PB, support, resistance
- Technical basics like candlesticks and trendlines
- Forex basics: pairs, pips, lots, sessions, risk management
